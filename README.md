<p align="left">📌 Project Overview<br><br>The Vehicle Rental Management System Backend API is a secure, scalable RESTful backend application built using Node.js, TypeScript, Express, and PostgreSQL.<br><br>It provides a complete backend solution for managing:<br><br>User authentication & authorization<br><br>Role-based access control (Admin & Customer)<br><br>Vehicle inventory management<br><br>Vehicle booking & rental operations<br><br>This system follows modular architecture, MVC pattern, and industry-standard backend practices.<br><br>🚀 Features<br>🔐 Authentication & Authorization<br><br>JWT based authentication<br><br>Secure password hashing using bcrypt<br><br>Role-based authorization (Admin / Customer)<br><br>👥 User Management<br><br>User registration & login<br><br>Admin-controlled user management<br><br>Secure user data handling<br><br>🚗 Vehicle Management (Admin Only)<br><br>Add new vehicles<br><br>Update vehicle details<br><br>Delete vehicles<br><br>View all vehicles<br><br>📦 Booking System<br><br>Customers can book vehicles<br><br>Automatic availability update<br><br>Booking history tracking<br><br>Admin can view all bookings<br><br>🛡 Security<br><br>JWT authentication<br><br>Password hashing using bcrypt<br><br>Role-based middleware protection</p>
